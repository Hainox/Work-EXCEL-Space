# CLAUDE.md — Work-EXCEL-Space

## Проект
Рабочее пространство для работы с Excel и табличными данными.

## Стандарты кода
- Чистый, документированный код
- Проверка входных данных из таблиц
- Обработка ошибок при чтении/записи файлов
- Никаких хардкодных путей или секретов

## Работа с данными
- Валидация всех входных данных перед обработкой
- Бэкап перед изменением оригинальных файлов
- Логирование операций с файлами
- Обработка кодировок (UTF-8, Windows-1251)

## Тесты
- Тестировать граничные случаи в данных
- Проверять поведение при пустых/некорректных файлах
